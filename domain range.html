Domain & Range of a Function

Understanding inputs and outputs using graphs and motion

The domain of a function is the set of all possible input values (x-values) for which the function is defined.

From a graph, the domain is obtained by projecting the curve onto the x-axis.

ðŸ“Š Range of a Function

The range of a function is the set of all possible output values (y-values) that the function can produce.

From a graph, the range is obtained by projecting the curve onto the y-axis.

ðŸ§  Key Observation

Every function has a domain, but the range depends on the behavior of the function. Different functions may have the same domain but very different ranges.
