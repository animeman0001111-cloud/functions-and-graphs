Operations on Functions

Algebraic operations performed on two or more functions

➕ Addition of Functions

If f(x) and g(x) are two functions, their sum is defined by adding their outputs for each input.

(f + g)(x) = f(x) + g(x)

The domain of (f + g) consists of all values of x that belong to the domains of both f and g.

➖ Subtraction of Functions

Subtraction measures how much one function differs from another at each input value.

(f − g)(x) = f(x) − g(x)

The domain is the set of values common to both functions.

✖️ Multiplication of Functions

The product of two functions is obtained by multiplying their outputs for each input value.

(fg)(x) = f(x) · g(x)

This operation is useful in modeling combined effects.

➗ Division of Functions

Division compares the outputs of two functions as a ratio.

(f / g)(x) = f(x) / g(x),   g(x) ≠ 0

The function is undefined wherever g(x) = 0.
